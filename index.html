¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para definir el estilo y el diseño visual de los documentos HTML. Permite controlar la apariencia de las páginas web, como colores, fuentes, márgenes, tamaños, posiciones, y otros aspectos visuales.

Principales caracteristicas

  • Separación de contenido y presentación: Permite mantener el HTML enfocado en la estructura del contenido, mientras que CSS gestiona el diseño y la apariencia.
  • Reutilización de estilos: Un archivo CSS puede aplicarse a varias páginas web, haciendo más fácil mantener y actualizar los estilos de un sitio completo.
  • Cascada: El "Cascading" en CSS se refiere a cómo los estilos se aplican y sobreescriben unos a otros según su orden, especificidad y jerarquía.
  • Responsive design: CSS puede adaptarse a diferentes tamaños de pantalla y dispositivos (mediante media queries), facilitando el diseño de sitios web responsivos.

Resumen

Es un lenguaje utilizado para describir la presentación visual de páginas web, controlando el diseño y la apariencia de los elementos HTML, como colores, tipografías, márgenes y posiciones. Permite separar el contenido (HTML) de su estilo, facilitando la gestión del diseño en sitios web y la creación de páginas web responsivas, adaptadas a distintos dispositivos.
